## 品牌特点

- **主色**：深海蓝 `#083A67` — 专业、值得信赖
- **Accent**：珊瑚橙 `#FF9B7D` — 活力、温暖
- **次色**：钢蓝 `#4F82BA` — 辅助视觉层级
- **风格**：专业而温暖，适合企业培训和咨询场景
