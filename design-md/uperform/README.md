## 品牌特点 — 4 层色彩体系

按国际设计标准（Material Design / IBM Carbon / Figma）采用 **PRIMARY / NEUTRALS / SECONDARY / ACCENT** 4 层结构，各司其职：

| 层级 | 色系 | 代表色 | 视觉量 | 职责 |
|------|------|--------|--------|------|
| **PRIMARY** | 蓝色 (5色) | 海洋蓝 `#1E4976` | 60–70% | 品牌识别 |
| **NEUTRALS** | 灰色 (5色) | 碳黑/石板/钢铁/银/雾白 | 功能性 | 文字/边/底 |
| **SECONDARY** | 金色 (2色) | 尊贵金 `#D4AF37` | 装饰性 | 高级感（≤ 3 处）|
| **ACCENT** | 橙色 (2色) | 活力橙 `#FF6B35` | 行动性 | CTA 按钮（≤ 3 处）|

**3 条使用铁律**：
- 🔴 橙色是"放大镜"，全场 ≤ 3 处
- 🟡 金色永远搭配深色，不放冰川蓝上
- ⚪ 文字层级只走灰阶，蓝/橙/金不参与正文

**典型场景组合**：
- 商务场景：深海蓝 + 尊贵金 + 活力橙 + 金底白字按钮
- 数字产品：雾白 + 香槟金 + 海洋蓝底白字按钮
- 营销物料：天空蓝 + 尊贵金 + 活力橙 + 金字
- 风格：深邃、尊贵、活力，适合企业培训、咨询与高端场景
