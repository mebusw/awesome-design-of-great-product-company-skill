02 / Four Critical Rules

4 条使用铁律

这 4 条铁律是整个色彩体系的灵魂——保证主色不被稀释、辅助色不被混用、渐变不被降级、高级感不被油漆感破坏。

  • 1🆕 辅助色二选一,不可混用 — 一套设计中只允许使用 方案 A（金色系） 或 方案 B（粉橙系） 中的 一种,不能同屏混用;不能两套同时出现。
  • 2辅助色永远搭配深色或纯白 — 金色 + 深海蓝 = 经典奢华;金色 + 雾白 = 轻柔优雅;粉橙 + 深海蓝 = 现代深色背景;粉橙 + 白色 = 温暖明亮。辅助色不要放在冰川蓝上（会糊）。
  • 3文字层级用灰色系统,蓝/辅助色不参与正文 — 正文只走 #1A1A1A → #3D4852 → #6B7785 的灰阶。蓝/辅助色是装饰色,不是文字色。
  • 4🆕 辅助色一律使用渐变,禁止纯色 — 无论是金属色（Gold/Golden Glow）还是温润色（Blush/Peach）,纯色 = 廉价 = 油漆感。所有辅助色元素(背景/底色/按钮/描边/文字)100% 渐变;辅助色字必须用 background-clip: text 渐变技术,绝不能 color: #EABA1D 或 color: #CE6E7B。
03 / Page Type Allocation

按页面类型分配 (三明治结构 · 二选一)

不同页面类型有不同的色彩主导——封面/章节/结尾用深海蓝主导,内容页用海洋蓝主导,数据/成果页让碳黑与辅助色数字承担视觉。辅助色一栏的「金色版」与「粉橙版」二选一,不可同屏。

06 / 🆕 辅助色字渐变 (铁律 #4 样本 · 二选一)

辅助色字渐变 — 深色背景上的金属/温润字

深色背景上的辅助色文字必须用 background-clip: text 渐变技术,让字本身呈现高级光泽。绝不能 color: #EABA1D 或 color: #CE6E7B 写死。下方案 A 与方案 B 二选一展示。

方案 A — 金色

UPerform · SIGNATURE COMBINATION (A)

海洋蓝 + Gold 金字

深海蓝铺底,金属质感的金色文字浮于其上,营造"深海藏金"的视觉张力。用于高端封面、Hero 标题、奖项大数字、VIP 标识、限量版标签。

方案 B — 粉橙

UPerform · SIGNATURE COMBINATION (B)

海洋蓝 + Blush 粉橙字

深海蓝铺底,温润柔和的粉橙文字浮于其上,营造现代感与亲和力。用于温暖场景、年轻向、女性向、亲和 CTA、限时活动。
