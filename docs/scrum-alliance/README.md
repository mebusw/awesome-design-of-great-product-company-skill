## 品牌特点

- **主色**：Deep Blue `#182340` — 主色调、主导色
- **第二主色**：Medium Blue `#375EDF` — 次级强调
- **浅色辅助**：Light Blue `#A4D5EB` — 点缀（不宜主导）
- **活力强调**：Orange `#FA9449` — CTA、强调（不宜主导）
- **背景色**：Off-White `#F3F3F3` — 简洁、专业
- **风格**：专业权威、清晰明亮，适合 Scrum 认证培训
